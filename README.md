# Tugas Besar Programming SEKURO 2023 Kelompok 2

Pada Tugas Besar Programming SEKURO 2023, cakru diminta membuat program sistem kapal yang mengimplementasikan object-oriented programming. Program dibuat dalam bahasa C++ dan menggunakan library iostream, stdlib.h, string, time.h dan math.h

### **Overview Program**
Program dimulai dengan menampilkan koordinat kapal cakru (0, 0) dan kapal musuh (random), health kapal cakru dan kapal musuh, jarak antara dua kapal, serta jarak tembak maksimum (range) kapal cakru dan kapal musuh. Program memberikan 3 pilihan perintah, yaitu pilihan 1 untuk menembak, pilihan 2 untuk bergerak, dan pilihan 3 untuk diam di tempat. Pengguna dapat menembak hanya jika jarak terhadap musuh kurang dari range tembak kapal cakru. Pengguna dapat menggerakkan kapal satu satuan ke arah atas/bawah/kiri/kanan. Program akan berhenti ketika health kapal musuh habis (nol atau negatif).

### **Cara Menjalankan Program**
Compile dan run program di directory ``TugasBesar_Programming_Kel_02/src``
